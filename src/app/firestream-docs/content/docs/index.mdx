# Welcome to Firestream

Firestream is a serverless data warehouse designed as "create-react-app for data engineering" - providing a fully local K3D cluster that can be deployed to production in GCP or AWS.

## Core Concepts

- **Local Development**: K3D (Kubernetes in Docker) mirrors production environments
- **Infrastructure as Code**: DevContainer with Docker-from-Docker pattern
- **Real-time Streaming**: Webhooks, websockets, REST with streaming effects
- **100% Open Source**: Built on Bitnami charts, Apache projects, K8s ecosystem
