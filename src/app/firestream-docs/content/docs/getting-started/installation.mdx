# Installation

## Using DevContainer (Recommended)

The easiest way to get started is using VS Code with the DevContainer extension:

1. Clone the repository
2. Open in VS Code
3. Click "Reopen in Container" when prompted

The DevContainer will automatically set up:
- K3D cluster on your host's Docker Engine
- All required tools and dependencies
- Port forwarding to services

## Manual Installation

If you prefer manual setup, you'll need:

- Docker Engine
- K3D
- kubectl
- Helm
- Nix (optional, for reproducible builds)
