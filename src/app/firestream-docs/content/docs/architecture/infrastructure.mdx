# Infrastructure

Firestream uses Kubernetes with Helm charts for service deployment.

## Components

- **K3D** - Local Kubernetes clusters
- **Helm Charts** - Service deployments (Bitnami fork)
- **Containers** - Custom Docker images

## Data Stack

- MinIO (S3-compatible storage)
- Delta Lake (table format)
- Apache Spark (distributed processing)
- Kafka (message streaming)
- Airflow (workflow orchestration)
