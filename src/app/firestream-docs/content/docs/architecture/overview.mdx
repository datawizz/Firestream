# System Overview

Firestream is a polyglot system combining multiple languages for optimal performance.

## Multi-Language Stack

1. **Rust** - TUI interface, API server, infrastructure tooling
2. **Python** - ETL library, Airflow DAGs, data processing
3. **Scala** - Spark stateful streaming applications
4. **Node.js** - Middleware services, dashboard frontends
5. **Nix** - Reproducible build environment
