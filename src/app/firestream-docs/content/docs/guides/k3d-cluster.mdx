# K3D Cluster

K3D provides lightweight Kubernetes clusters running in Docker.

## Cluster Operations

```bash
# Create cluster
make development

# Resume existing cluster
make resume

# Clean rebuild
make development_clean
```

## Resource Requirements

- Minimum: 4 CPUs, 8GB RAM
- Recommended: 8 CPUs, 16GB RAM
