# Contributing

We welcome contributions to Firestream!

## Development Workflow

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests
5. Submit a pull request

## Code Style

- Rust: Follow rustfmt defaults
- Python: Follow PEP 8
- TypeScript: Use ESLint configuration
